# Sistema de Producción Textil

Sistema desarrollado en Spring Boot para gestionar la producción de rollos de tela a partir de hilo enviado por clientes.

## Tecnologías

- Java
- Spring Boot
- Spring Data JPA
- Maven
- MySQL

## Módulos implementados

- Cliente
- Hilo
- OrdenProduccion
- DetalleOrdenProduccion
- RolloTela
